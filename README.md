## AI-Powered Vulnerability Management & Technical Support Assistant

Enterprise-grade IT operations dashboard that unifies Level 1 and Level 2 service desk automation, vulnerability monitoring, endpoint health, and live synchronization with ServiceNow, Jira, and Freshservice.

### Features
- Intelligent ticket queue with SLA awareness, AI diagnostics, and escalation guardrails.
- Vulnerability radar prioritizing Windows, macOS, and network surface exposures with remediation guidance.
- Endpoint health telemetry fed by Intune, Jamf, and network probes with compliance scoring.
- Integration heartbeat with ServiceNow, Jira, Freshservice, Intune, and Jamf.
- Automation workflows for user onboarding/offboarding and quarterly access reviews.
- Real-time activity feed highlighting AI-driven incident actions and security enforcement.

### Getting Started
```bash
npm install
npm run dev
```

Navigate to [http://localhost:3000](http://localhost:3000) to explore the dashboard.

### Production Build
```bash
npm run build
npm start
```

The app is ready for deployment on Vercel without additional configuration.
